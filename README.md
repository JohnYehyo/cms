# CMS



## 项目介绍

> Springboot+SpringSecurity+jwt+redis+mybatis-plus



## 项目结构

.
├── pom.xml
├──doc                        ----------------------- 文档、sql脚本
├── rjsoft-common   ----------------------- 常用模块包含通用工具类等
├── rjsoft-core    	   ----------------------- 模型层
├── rjsoft-dao            ----------------------- 持久层
├── rjsoft-service      ----------------------- 业务层
└── rjsoft-web           ----------------------- web层 controller、配置、aop、filter、interceptor等
